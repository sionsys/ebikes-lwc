# User Story Specification: Manufacturing Year Filter ## Requirements - Add a custom field `Year__c` (Number, 4, 0) to `Product__c`. - Expose the field on the `productCard` component. - Add a numeric filter input for Model Year on `productFilter`. ## Implementation Strategy - Metadata deployment via standard Salesforce project layout structure. - State mutation capture via `handleYearChange` inside the component architecture. - Publish changes using the built-in message channel context wrapper.
